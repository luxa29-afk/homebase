Welcome to HomeBase
Uses Firebase — free, instant, works everywhere. One-time setup in about 3 minutes.
1
Create a free Firebase project
Go to console.firebase.google.com → Add project → give it any name → disable Google Analytics → Create project.
Open Firebase Console →
2
Create a Realtime Database
In your project: Build → Realtime Database → Create Database → choose any region → Start in test mode → Enable.
3
Paste your Database URL here
Copy the URL shown at the top of the Realtime Database page. It looks like https://your-project-default-rtdb.firebaseio.com
Test mode is fine for personal use — it allows read/write without login. Both you and Tino use the same URL. Your Firebase URL is stored only in each browser locally.
